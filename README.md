# GenAI Job Recommendation System

This project is an AI-powered job recommendation system that uses Large Language Models (LLMs) and vector embeddings to match users with the most relevant job opportunities. It analyzes user profiles, skills, and job descriptions to generate personalized and accurate career recommendations.
